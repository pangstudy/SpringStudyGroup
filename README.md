### Repo 내려받기
+ 자신이 원하는 위치로 이동
+ git clone으로 해당 repository를 내려받기
```
git clone https://github.com/pangstudy/SpringStudyGroup.git
```
+ 디렉토리 안으로 이동
```
cd SpringStudyGroup
```
+ 자신의 브랜치를 만든다
    > git checkout -b 브랜치이름
+ 학습한 내용을 add, commit한다
+ 서버에 push
```
git push origin 브랜치이름
```
